# GIT-GITHUB

Este repositorio contiene los ejercicios desarrollados de manera individual durante el curso de Git y GitHub, con el objetivo de fortalecer los conocimientos básicos sobre control de versiones y el uso de herramientas para la gestión y seguimiento de cambios en proyectos de software.

En el repositorio se aplican conceptos como inicialización de repositorios, manejo de commits, ramas y control del historial de versiones.
